# ClarityIQ – AI Notes Humanizer

ClarityIQ is an AI-powered tool that converts robotic or textbook-style notes into natural, human-readable explanations.

## Features
- Simple note humanization
- Simplified vocabulary output
- Gen Z-style rewriting mode
- Cause–effect explanation mode

## Tech Stack
- HTML
- JavaScript
- JSON
- Groq API
- Vercel (deployment)
